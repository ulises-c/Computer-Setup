# Useful Windows software

## Utilities

1. SoundSwitch | [Download](https://soundswitch.aaflalo.me/#) | [GitHub](https://github.com/Belphemur/SoundSwitch)
   1. Switch playback and recording devices using hotkeys — useful when switching between headphones, speakers, and monitors
2. PowerToys | [Download](https://learn.microsoft.com/en-us/windows/powertoys/) | [GitHub](https://github.com/microsoft/PowerToys)
   1. Microsoft utility suite for power users — includes FancyZones (window snapping), PowerToys Run (launcher), and more
